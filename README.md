# Generador de Subtítulos con IA

Esta herramienta permite buscar subtítulos en OpenSubtitles, descargar versiones en inglés, traducirlas al español usando IA (OpenAI) y subirlas automáticamente.

## Configuración Rápida

1.  Abre el archivo `.env.example`, renómbralo a `.env` (o deja que el script lo cree) y rellena tus claves.
2.  Ejecuta el script de inicio automático:
    ```bash
    ./start.sh
    ```
    Este script creará el entorno virtual, instalará las dependencias y lanzará el servidor.

3.  Abre `http://localhost:8000` en tu navegador.

## Configuración Manual (Opcional)

Si prefieres hacerlo manualmente:

1.  Copia `.env.example` a `.env` y rellena tus credenciales.
